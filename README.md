\## 📌 About the Project

Fully responsive café website built with pure HTML, CSS and JavaScript. No frameworks used.



\## 🛠️ Tech Stack

\- HTML5, CSS3, JavaScript ES6+

\- Google Fonts

\- Git, GitHub, GitHub Pages



\## ✨ Features

\- Responsive design

\- Filterable menu (Drinks, Food, Seasonal)

\- Table reservation form with validation

\- Gallery with lightbox

\- Animated counters and scroll reveal

\- Mobile hamburger menu



\## 📁 Project Structure

index.html, css/style.css, js/data.js, js/menu.js, js/form.js, js/animations.js, js/main.js



\## 🚀 Run Locally

git clone https://github.com/ShrijeetaDas006/brewed-awakenings.git

cd brewed-awakenings

python -m http.server 8000

Open http://localhost:8000
